## 2025.02.19 - 长度为 n 的开心字符串中字典序第 k 小的字符串

### 解题思路
回溯法

生成所有可能的字符串，存入列表中。如果列表长度小于k，则返回空字符串，否则返回第 $k$ 个（索引为 $k-1$）字符串

```java
class Solution {
    private List<String> res = new ArrayList<>();

    public String getHappyString(int n, int k) {
        generate(n, "");

        if (res.size() < k) {
            return "";
        }

        return res.get(k - 1);
    }

    private void generate(int n, String curr) {
        if (curr.length() == n) {
            res.add(curr);
            return;
        }

        for (char i = 'a'; i <= 'c'; i++) {
            if (curr.length() > 0 && curr.charAt(curr.length() - 1) == i) {
                continue;
            }
            generate(n, curr + i);
        }
    }
}
```

### 优化算法

按照字典序生成合规的字符串，在找到第 $k$ 个时，终止生成，直接返回结果

```java
class Solution {

    int len, K;
    int total = 0;
    String res = "";

    public String getHappyString(int n, int k) {
        len = n; K = k;
        StringBuilder s = new StringBuilder();

        for(char i = 'a'; i <= 'c'; i++) {
            if(dfs(0, s.append(i))){
                return res;
            } 
            s.deleteCharAt(s.length() - 1);
        }

        return "";
    }

    private boolean dfs(int index, StringBuilder s) {
        if(index + 1 == len) {
            total++;
            if(total == K) {
                res = s.toString();
                return true;
            } 
            return false;
        }

        for(char i = 'a'; i <= 'c'; i++) {
            if(s.charAt(index) != i) {
                if(dfs(index + 1, s.append(i))) return true;
                s.deleteCharAt(s.length() - 1);
            }
        }

        return false;
    }
}
```
